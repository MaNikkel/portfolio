A few projects:

Instagram clone

Destined to practice and learn concepts in node.js, react and react-native, this project implements a basic clone of Instagram, in web and mobile application.

It implements post uploading (in both environments) real-time feed visualization using Websockets protocol and post likes, using the same protocol.

Using a NoSQL database (MongoDB) this project uses multipart form-data and filesystem structure, that is for the app loading time optimization, considering multi-users logged-in.

Laravel Blog

Idealized to use the Laravel framework, this project implements a complete blog application, using pure PHP and Laravel resources, like tinker, controllers, auth, migrations and more.

This app implements a single account, with an complete crud application for the user posts, but it also implements a feed visualization of posts of all users, just like a normal blog application does.

MySQL was the chosen database, due to their easy usage with PHPMyAdmin, but all their management was made via artisan migrations, making the database manipulation much easier.

PROJECT

Line Follower Screen

In this project, I wish to read incoming data, sourced from my Arduino based line follower robot, and write it into a database so I could use it to analyze it and build some cool graphs. To that, I have used the Java language, due to their facilities to build desktop interfaces using Swing and their pre-loaded MySQL Driver (at least in Netbeans IDE).

Beside the computer/Arduino communication issues at the beginning, that software worked just fine for me, helping me out to easily gather all information from the controller implemented in that robot, saving me a lot of time.
